## 共通の手順

1. プロジェクトディレクトリに移動
2. `npm install` で依存関係をインストール
3. `npx tsc` でTypeScriptをコンパイル
4. ブラウザで `index.html` を開く

## 開発モード

自動コンパイルモードで開発する場合：
```bash
npx tsc -w
```

このコマンドを実行すると、ファイルを保存するたびに自動でコンパイルされます。
