# HTML/CSS基礎

フロントエンド開発の第一歩として、HTML（HyperText Markup Language）とCSS（Cascading Style Sheets）を学びましょう。

## HTMLとCSSとは？

### HTML - Webページの骨格

HTMLは、Webページの**構造や内容**を定義する言語です。見出し、段落、リスト、画像、リンクなど、ページに表示される要素を記述します。

例えば：
- 「これは見出しです」
- 「これは段落です」
- 「ここに画像を表示します」

といった**情報の意味**をブラウザに伝える役割を持っています。

### CSS - Webページの見た目

CSSは、HTMLで記述した要素の**デザインやレイアウト**を指定する言語です。色、フォント、配置、アニメーションなど、見た目に関するあらゆるスタイルを制御します。

例えば：
- 「この見出しを青色にして、大きくする」
- 「この段落の背景を灰色にする」
- 「この要素を画面の中央に配置する」

といった**視覚的な表現**を定義します。

## なぜHTML/CSSが重要なのか？

1. **すべてのWebサイトの基礎**
   どんなに複雑なWebアプリケーションも、最終的にはHTMLとCSSでユーザーに表示されます。

2. **他の技術を学ぶための土台**
   JavaScriptやフレームワーク（React、Vueなど）を学ぶ際も、HTML/CSSの知識は必須です。

3. **デザインを実装できる力**
   自分のアイデアを形にしたり、デザイナーの作ったデザインをコードで再現したりする力が身につきます。

以下のZenn本で、HTML/CSSの基礎をしっかり学びましょう。この教材は無料で、初心者にもわかりやすく書かれています。

**[HTML/CSS入門](https://zenn.dev/ojk/books/intro-to-html-css)**

### 学習のポイント

- **コードはコピペでOK、でも全て理解することが大切**
  教材のコードをコピー&ペーストして試すことは全く問題ありません。むしろ推奨します。ただし、コピペしたコードが「何をしているのか」「なぜそうなるのか」を理解することが重要です。動くだけで満足せず、一行一行の意味を考えましょう。

- **Live Previewで即座に確認**
  VS Codeで`.html`ファイルを作成したら、Live Preview拡張機能を使ってブラウザで表示を確認しましょう。コードを書いたら即座に結果が見えるので、理解が深まります。Live Previewの使い方は[拡張機能のセットアップ](../environment/live_preview)で確認できます。

- **覚えるのではなく、理解すること**
  HTMLタグやCSSプロパティを全て暗記する必要はありません。「こういうことができる」という知識を持っておけば、必要なときに調べて使えます。大切なのは、「どういうものがあるか」を一通り学んで知識として持っておくことです。

- **ブラウザはGoogle Chromeを使うこと**
  Web開発では、Google Chromeを使うことを強く推奨します。開発者向けのツールが充実しており、多くの開発者が使用しているため、困ったときの情報も豊富です。

- **デベロッパーツールを活用**
  Chromeのデベロッパーツールを使うと、実際のWebサイトがどのように作られているか見ることができます。開き方や使い方は以下の記事を参考にしてください。

  **[Chrome デベロッパーツールの使い方](https://willcloud.jp/knowhow/dev-tools-01/)**

  実際のWebサイトを見ながら「このデザインはどうやって作られているんだろう？」と調べてみると、学びが深まります。

## 次のステップ

HTML/CSSの基礎を理解したら、次は**JavaScript**を学んで、Webページに動きをつけられるようにしましょう。

JavaScriptを学ぶことで、ユーザーの操作に反応したり、データを動的に表示したりする、インタラクティブなWebページを作れるようになります。
