# Node.jsのインストール

TypeScriptを学ぶためには、**Node.js**という実行環境が必要です。Node.jsをインストールすることで、JavaScriptやTypeScriptをコンピュータ上で実行できるようになります。

## Node.jsとは？

Node.jsは、ブラウザの外でJavaScriptを実行するための環境です。これを使うことで、以下のことができるようになります：

- TypeScriptやJavaScriptのコードをコンピュータ上で実行する
- npmというパッケージマネージャーを使って、便利なライブラリをインストールする
- サーバーサイドのプログラムを開発する

## インストール方法

### **重要な注意点**

インストール方法は、使用しているOSによって異なります。

#### Macユーザーの方

Mac環境に直接Node.jsをインストールします。

#### Windowsユーザーの方

**Windows環境ではなく、Ubuntu（WSL）のLinux環境**にNode.jsをインストールします。

なぜなら、Ubuntu/WSLは**Linux環境**であり、開発ツールの多くがLinux用に最適化されているからです。Windows環境に直接インストールすると、後で問題が起きる可能性があります。

必ずUbuntu（WSL）のターミナルを開いて、以下の手順を実行してください。

## インストール確認

Node.jsが正しくインストールされたか確認しましょう。ターミナル（Macはターミナル、WindowsはUbuntu）で以下のコマンドを実行してください。

```bash
node --version
```

バージョン番号（例：`v20.11.0`）が表示されれば、インストール成功です。

```bash
npm --version
```

こちらもバージョン番号（例：`10.2.4`）が表示されれば、npm（パッケージマネージャー）も正しくインストールされています。

## トラブルシューティング

### コマンドが見つからないと表示される

- ターミナルを一度閉じて、開き直してください
- Windowsの方は、Ubuntu（WSL）のターミナルで実行しているか確認してください

### インストールがうまくいかない

- インストール手順を最初から見直してください
- エラーメッセージをよく読んで、何が問題なのか確認してください
- すでに実行した手順を再度実行していないか確認してください
